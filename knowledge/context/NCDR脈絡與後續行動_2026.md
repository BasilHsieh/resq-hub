# NCDR 脈絡與後續行動筆記（2026-05 版）

## NCDR 計畫脈絡時序（截至 2026-05 已知）

| 階段 | 年份 | 名稱 |
|---|---|---|
| 緣起 | 2003 | EMIS（應變管理資訊系統） |
| 轉折 | 2009 | 莫拉克風災 |
| 法規修正 | 2010 | 災防法修法、地方須設災防辦 |
| 先期計畫 | 2015-16 | 中央與地方防救災情資整合研究先期計畫（12 學研機構） |
| 試辦 | 2017 | 試辦計畫（擴張到 22 學研機構） |
| 正式 | 2017-2019 | 中央與地方防救災情資整合管理研究計畫（3 年期） |
| **後續** | **2020-2026** | **待確認**（NCDR 官網未有明確說明） |

**已知 2026 NCDR 整體方向**（2026-05 官網查詢）：
- 「災防資訊整合加值應用」仍是主軸
- 高低頻極端氣象災害新預警技術研發
- 國際合作（IWCC 2026 氣候調適會議）
- 但「中央與地方防救災情資整合管理計畫」是否仍在運作未明
- 嘗試查訪官網 https://cldpmo.ncdr.nat.gov.tw/ 兩次都連線超時——站台可能已停用或極不穩定

## 若 Basil 想接觸花蓮對口劉瑩三

### 為什麼劉瑩三可能值得接觸

1. **位置精準**：他是 NCDR 計畫架構下花蓮縣的學研機構對口，2017 加入
2. **2025 花蓮光復**：劉瑩三所在的東華大學就在花蓮，2025 馬太鞍溪堰塞湖事件他大概率有參與或觀察
3. **方法論**：他的角色是「協助地方政府提升量能」，跟 ResQ Hub 想處理的層次正好是**互補的另一端**（他做的是政府/學研端，ResQ 想做的是民間/散兵端）
4. **基本背景**：地質、災害管理專業

### 接觸前該準備的事

不要直接寫信問「能否合作」。先：

1. **Google 一下劉瑩三老師近期發表**——確認他的學術重心是否有移動
2. **了解東華大學是否仍在 NCDR 計畫架構**——可能透過詢問東華大學自然資源與環境學系或地科系
3. **準備一個具體問題**——不要開放式問「您怎麼看 ResQ Hub」，要問具體的，比如：
   - 「2025 馬太鞍溪事件期間，您所在的學研機構是否參與災情研判？運作機制是怎樣？」
   - 「您觀察到的民間救援與政府/學研端之間，最大的協作缺口是什麼？」
   - 「您是否認為民間端有可能成為情資整合的補充？」

### 預期可能的反應

正面情境：
- 劉老師願意分享現場觀察與制度脈絡
- 可能透露 NCDR 計畫的近況、東華大學的內部討論

負面情境：
- 學研機構在計畫架構內，可能對「民間平台」這類想法保持距離（涉及學術中立、不便評論民間倡議）
- 計畫如已結束或他已不在計畫中，他可能無法提供有效資訊

**最壞情況也有收穫**：至少能確認 NCDR 計畫的現況。

## 替代/輔助接觸對象

如果劉瑩三那條路不順，其他可能值得探索：

1. **林培安**（NCDR 數位化平台論文 2019 作者，NTU 地理所校友）
   - 他研究的就是 NCDR 計畫的運作機制
   - 校友身份較好聯絡（NTU 地理所、周素卿老師研究團隊）
   - 適合問：論文 2019 完成後 NCDR 的後續發展、他現在的工作（學界 or 業界）

2. **鄭宇君**（玄奘大學大眾傳播學系，2014 社交媒體公民參與論文作者）
   - 是學界少數深入研究莫拉克民間災情網站的人
   - 跟 ResQ Hub 的議題最近
   - 可以問她對 2014 後台灣公民資訊平台演化的觀察

3. **Xdite（鄭伊廷）**（莫拉克災情支援網架站者）
   - 雖然她現在主要做技術創業（OurForce 等）跟災害領域距離較遠
   - 但她是「2009 年實作過 ResQ Hub 想做的事」的人
   - 即使她不參與，她的經驗教訓直接對應 ResQ Hub 的核心問題

4. **林萬億**（國策顧問、社工學界泰斗，2025 想想花蓮演講者）
   - 制度層級的權威，但相對難接觸
   - 若 Basil 想在公部門產生槓桿，這條線最有效
   - 通常需透過學界引介
