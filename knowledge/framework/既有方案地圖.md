# 既有方案地圖

> **用途**：避免重複造輪子的依據；定位 ResQ Hub 不該跟誰競爭。
> **核心觀察**：台灣有不少方案，但有 2 個結構性空白：(1) NGO 之間 ↔ 散兵志工 ↔ 受災民眾 的橫向協作層；(2) 災後（非災時）階段的民間協作。ResQ Hub 切入這兩個空白。

## 政府主導的方案

### ① NCDR「中央與地方防救災情資整合管理研究計畫」(2015-)

**做什麼**：
- 整合中央各部會的災害情資到災害情資網
- 透過 22 個地方學研機構作為「中央 ↔ 地方」橋樑
- 提供地方版災害情資網、整合儀表板、災害兵棋台
- 推動降尺度科研、教育訓練
- 創造跨域學習場域

**不做什麼**：
- 不接受公民回報（「恐參雜假新聞」）
- 不涉及 NGO 之間協作
- 不處理散兵志工
- 不涉及災後重建期民間協作

**為何留下空白**：
- 政策設計就排除民間端（不是技術做不到）
- 假設政府是資訊主體，民眾是資訊接收者
- 學研機構橋樑只到地方政府為止，不到 NGO/民間

**ResQ Hub 不該做什麼**：
- 不重複建一個資訊整合平台給政府用
- 不主打「給政府決策的情資」
- 不對接 NCDR 計畫架構（除非主動合作）

**ResQ Hub 該做什麼**：
- 接下 NCDR 不做的那層（NGO ↔ 散兵 ↔ 受災民眾）

---

### ② 強韌台灣大規模風災震災整備與協作計畫（2022-2026）

**做什麼**：
- 推動「防災士 + 韌性社區 + 區域聯防」三位一體
- 強化縱向（中央 → 地方 → 社區）協作
- 投入跨部會、跨層級的政策整合
- 5 年期、167 頁的完整制度設計

**不做什麼**：
- 整份計畫 167 頁**無「散兵志工」概念**
- 「公所為民間協作樞紐」設定在公所自身受災時失效（花蓮光復案例）
- 民間 NGO 在計畫中角色模糊

**為何留下空白**：
- 預設「組織化志工」是唯一健康模式
- 預設地方政府量能可以撐起樞紐角色
- 沒有針對「自發志工湧入」的設計

**ResQ Hub 不該做什麼**：
- 不重複「防災士培訓」這條路（這是政府的）
- 不主張取代「韌性社區」概念

**ResQ Hub 該做什麼**：
- 補強韌台灣計畫**承認但沒解**的橫向協作層
- 為散兵志工設計（強韌台灣沒做）

---

### ③ 災害防救法 + 緊急命令框架

**做什麼**：
- 三級防災體系（中央 → 直轄市/縣市 → 鄉鎮市）
- 各級災害應變中心開設機制
- 軍隊支援、消防主導
- 復原重建程序

**不做什麼**：
- **中央依法不能接管**地方應變（謝志強 2014）
- 災害現場協調管理系統未在法律規定
- 公民/NGO 角色定位模糊
- 災害應變中心是任務編組而非專責

**為何留下空白**：
- 法律是 2000 年制定（之後修改但結構未變）
- 修法需政治決定
- 中央地方權力分配卡關

**ResQ Hub 該做什麼**：
- **這個權力真空就是民間補位的法理空間**
- ResQ Hub 不是補政府的位置，是補政府法理上不能填的位置
- 這也是 ResQ Hub「為什麼是民間」的法理依據

## 民間/NGO 主導的方案

### ④ 慈濟動員模型

**做什麼**：
- 跨組織動員（全台慈青、志工幹部、慈濟基金會職員）
- 標準化作業流程（撤離、安置、發放）
- 物資募集與分配（藍天白雲制服 → 識別 → 信任）
- 長期駐點（莫拉克後到八八永久屋持續多年）

**不做什麼**：
- 不收散兵志工為自家成員（雖然承認他們更主動）
- 不跟其他 NGO 共享資源（募款競爭）
- 不太處理「組織外人員需要協助」的情況

**揭露的代價**（慈濟莫拉克 2013）：
- 5:1 動員/服務比（5 個志工才能讓 1 個學生被服務）
- 隱性勞動：夜間 11 點寫日誌
- 宗教文化張力（強調「不喊苦」）

**ResQ Hub 不該做什麼**：
- 不要試圖整合進慈濟系統（慈濟有自己的系統）
- 不要對 U2 組織化志工為主用戶

**ResQ Hub 該做什麼**：
- 提供慈濟之外的「散兵志工 + 小型 NGO」一個協作場域
- 可能間接成為**慈濟看到「外部生態」的窗口**

### ⑤ 2009 莫拉克民間災情網站（Xdite / Billy Pan / 數位文化協會）

**做什麼**：
- **Xdite「莫拉克災情支援網」**：留言板形式，群眾自治篩選；9000+ 貼文，90% 匿名
- **Billy Pan「莫拉克災情地圖」**：地理視覺化；從 Plurk 抓災情訊息
- **數位文化協會「莫拉克民間災情網路中心」**：較專業的協作社群

**做對什麼**：
- 零身份門檻
- 重複容忍
- 群眾守版（gatewatching）
- 快速架站（一晚上）

**做錯什麼/限制**：
- 一次性，事件後消散
- 90% 匿名導致事後難分析、難維繫
- 依賴架站者個人英雄
- 「萬人按讚一人響應」的長期動員疲憊

**為何留下空白**：
- 沒有人接手成為長期平台
- 工具本身沒做成可複製的產品
- 失敗模式（短期熱血）未被解決

**ResQ Hub 該做什麼**：
- **直接繼承 Xdite 範式**（見 設計原則.md）
- 但加上「下一個 Xdite 不用從零開始」的可複用工具層
- 解 Xdite 沒解的「熱血退散後怎麼辦」問題

**ResQ Hub 不該做什麼**：
- 不要重複 Xdite 已踩過的坑（如複雜註冊、強制地圖）

---

### ⑥ 防災士制度（強韌台灣計畫子計畫）

**做什麼**：
- 個人化防災培訓（一般民眾受訓 → 取得「防災士」資格）
- 社區層級的災前準備能力

**不做什麼**：
- 規模有限（截至 2025 數萬人，遠低於需求）
- 災時/災後協作仍靠各自運作
- 沒有橫向協作工具

**ResQ Hub 對 ⑥ 的角色**：
- 防災士可能是 ResQ Hub 的早期種子用戶（受過訓、有意願）
- 但不能只服務防災士（會自我限縮）

## 國際參照

### ⑦ 美國 FEMA + 國家應變架構 (NRF)

**做什麼**（謝志強 2014、鄭阡妤 2014 詳述）：
- 總統可宣布「重大災害」→ 州長職權暫時凍結 → FEMA 介入
- 國家事故管理系統 (NIMS) 一元化指揮
- 災害現場聯合作業中心 (JFO) 整合聯邦/州/地方
- 整合民眾、企業、NGO、志工組織
- 災害現場指揮系統 (ICS) 標準化現場管理

**做對什麼**：
- 法律授權中央接管，責任歸屬清楚
- 整合志工組織有制度（如 VOAD 志工組織協會）
- 現場指揮系統可複製到任何災種

**台灣為何學不來**：
- 涉及中央地方權力分配修法
- 政治化讓「專業指揮政治讓位」難實踐

**ResQ Hub 可借鑑**：
- 「整合志工組織」的協作架構（VOAD 模式可參考）
- 「現場指揮」的標準化思維（如何讓現場節點互通）

---

### ⑧ 日本災害對策基本法

**做什麼**（謝志強 2014 詳述）：
- 內閣府中央防災會議統籌
- 設「特命防災擔當大臣」專責防災行政
- 三級體系（中央 → 都道府縣 → 市町村），市町村長有廣泛權限
- 大災時內閣總理可設「緊急災害對策本部」
- 中央可「指示」地方應變作為（vs 台灣只能「支援協調」）

**做對什麼**：
- 防災行政有明確主管
- 中央可指示地方（介於美國的接管與台灣的協調之間）
- 每年地方政府舉辦防災演習常態化（楊永年 2006 觀察）

**台灣為何學不來**：
- 政府文化差異
- 法律修正進度慢

**ResQ Hub 可借鑑**：
- 「常態化演習」可能是長期願景的一部分（讓 ResQ Hub 平時就有用，不只災時）

## 各方案的空白疊圖

把所有既有方案列出後，明顯看出兩個空白：

```
                   災前        災時        災後
              ┌─────────┬─────────┬─────────┐
   政府/學研  │ NCDR 計畫，強韌台灣，防災士 │
              ├─────────┼─────────┼─────────┤
   大型 NGO   │  慈濟、紅十字會（各自為政）  │
              ├─────────┼─────────┼─────────┤
   民間平台   │  Xdite 2009（一次性）     │
              ├─────────┼─────────┼─────────┤
   散兵 + 受災│   ❌ 沒有方案  ❌ 沒有方案  │  ← ResQ Hub 切入點
              └─────────┴─────────┴─────────┘
```

兩個空白：
1. **「散兵志工 + 受災民眾」這一列**：完全沒有結構性方案
2. **「災後」這一行**：政府方案重「災時應變」，民間方案重「災時資訊」，**災後幾天到幾週的協作期幾乎沒有專屬方案**

## ResQ Hub 的定位釐清

從既有方案地圖看：

✅ **該做的**
- 散兵志工 + 小型 NGO 的協作層（沒人在做）
- 災後協作期（沒人專注）
- 補 LINE 的盲點（沒人在做）

❌ **不該做的**
- 重做 NCDR（政府已做，且民間做沒用）
- 重做強韌台灣（政府已做）
- 跟慈濟競爭（規模不對等且文化不同）
- 重複 Xdite 2009 已試過的所有東西
- 試圖修改災害防救法或政府架構

⚠️ **可探索但小心**
- 跟學研機構合作（潛在橋樑，但他們屬於 NCDR 體系）
- 跟防災士社群合作（種子用戶，但規模有限）
- 在花蓮光復後跟林萬億等學者交流（學術正當性，但要小心政治化）
